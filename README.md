# NovSpringBoot
🎖A traditional framework with Spring Boot , applicable to interface.

#### SpringBoot框架整合
> Spring Boot Version 1.5.9.RELEASE

#### 关于
- 名字也要介绍下
	- 因为是2017年11月份整合的框架，所以就叫NovSpringBoot啦
- 毕业后学习并整合了一款较为成熟的Spring Boot框架
	- 轻量，优美，易用
- 适用于前后端分离项目
    - 可视化Swagger接口文档
    - 可视化Druid控制台
